# nhatrang-housing-analysis
# Анализ цен на жилье в Нячанге
Мой первый EDA-проект. 
   
Что сделал:
1. Загрузил датасет с 50+ объявлений о продаже недвижимости
2. Почистил данные, посмотрел распределение цен
3. Нашел самые дорогие и дешевые районы города
4. Стек: Python, Pandas, Matplotlib, Seaborn
